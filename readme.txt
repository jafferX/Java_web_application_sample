The application is built from a java backend running tomcat and can be
compiled in Eclipse or through maven. both the client and server must be run before application will work.

To start the server cd into the demo folder and run the command './mvnw spring-boot:run'
To start the client cd into the frontend folder and run the command 'ng serve'

The frontend can be accessed at the location localhost:4200.
The tomcat server lies on localhost:8080 and functions as a backend populating the frontend.
